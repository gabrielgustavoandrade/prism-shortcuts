# Privacy Policy

Prism Shortcuts runs locally in Chrome and does not collect, sell, or transmit personal data.

The extension uses Chrome storage only for local user preferences, such as whether clean URL copying is enabled. It reads the active tab URL and title only when the user invokes a shortcut that needs that information, such as copying the URL or creating a Markdown link.

Clipboard access is used only in response to user actions that explicitly copy content.
